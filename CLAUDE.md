# sample-data-room-app

## Agent skills

### Issue tracker

Issues and PRDs are tracked as GitHub issues in `jzsiggy/sample-data-room` (via the `gh` CLI). See `docs/agents/issue-tracker.md`.

### Triage labels

Five canonical triage roles using default strings (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context layout — one `CONTEXT.md` at the root is the glossary for the whole app (frontend + backend share one ubiquitous language). See `docs/agents/domain.md`.
